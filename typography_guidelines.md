# 워플로지 타이포그래피 시스템 가이드라인 (Typography Guidelines)

본 문서는 주식회사 워플로지(Worflogy)의 웹사이트 및 디지털 제품군에 공통으로 적용되는 타이포그래피 규칙을 정의합니다. 브랜드의 신뢰도 높은 인상과 현대적인 기술력, 세련된 소통의 정체성을 일관되게 전달하기 위해 설계되었습니다.

---

## 1. 폰트 패어링 시스템 (Font Pairing)

워플로지는 브랜드 가치를 시각적으로 반영하기 위해 **세리프(Serif)**와 **산세리프(Sans-serif)**의 대비를 정밀하게 구성합니다.

| 폰트 이름 | 분류 | CSS 변수 토큰 | 디자인 의도 및 주요 용도 |
| :--- | :--- | :--- | :--- |
| **Noto Serif KR** | 국문 세리프 | `--font-serif-ko` | - **신뢰와 깊이**: 국문 타이틀 및 핵심 헤드라인에 적용하여, 진중하고 사려 깊은 인상을 전달합니다. |
| **DM Serif Display** | 영문 세리프 | `--font-serif-en` | - **감성적 터치**: 연도, 섹션 번호, 보조 라벨 등 포인트 성격의 영문 요소에 이탤릭(Italic)으로 사용하여 고전적이고 세련된 디테일을 더합니다. |
| **Outfit** | 영문 산세리프 | `--font-sans-en` | - **현대성과 기술**: 브랜드 로고, 버튼, 수치 메타데이터 등 기하학적인 명료함과 현대적 기술 스타트업 이미지가 필요한 영역에 1순위로 사용합니다. |
| **Pretendard Variable** | 국문 본문 | `--font-body` | - **극대화된 가독성**: 긴 설명, 카드 내용 등 일반 텍스트의 고해상도 가독성을 완벽하게 제공합니다. (영문 혼용 시 **DM Sans**와 결합) |

---

## 2. 용도별 타이포그래피 규격 (Typography Scale)

모든 텍스트 요소는 아래 정의된 **9가지 용도별 규격**에 부합해야 합니다. 새로운 컴포넌트를 설계할 때도 반드시 이 범주 내에서 스타일을 부여하십시오.

```css
/* 예시: CSS 스타일 적용법 */
.my-new-title {
  font-family: var(--font-serif-ko);
  font-size: 38px;
  font-weight: 700;
  line-height: 1.3;
  letter-spacing: -1px;
}
```

### [규격표]

| 분류 (Class) | 적용 대상 (Targets) | 폰트 패밀리 | 굵기 (Weight) | 크기 (Size) | 행간 (Line Height) | 자간 (Letter Spacing) |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| **H1. 초대형 제목** | 메인 랜딩 히어로 타이틀 | `--font-serif-ko` | `700` | `60px` | `1.2` | `-2px` |
| **H2. 대형 제목** | 일반 페이지 헤더 타이틀 | `--font-serif-ko` | `700` | `42px` | `1.3` | `-0.5px` |
| **H3. 중형 제목** | 섹션 타이틀 | `--font-serif-ko` | `700` | `38px` | `1.3` | `-1px` |
| **H4. 소형 제목** | 카드 제목, 타임라인 제목, 피처 제목 | `--font-serif-ko` | `600` | `16px` | `1.4` | `-0.3px` |
| **Body. 기본 본문** | 일반 문단 (p), 레이아웃 기본 | `--font-body` | `400` | `16px` | `1.65` | `normal` |
| **Body-Sub. 설명** | 히어로 서브글, 섹션 설명글 | `--font-body` | `400` | `17px` | `1.85` | `normal` |
| **Body-Min. 초소형** | 카드 설명, 피처 설명 등 상세글 | `--font-body` | `400` | `13px` | `1.7` | `normal` |
| **Point Label. 포인트**| 섹션 카테고리 라벨, 연도, 비전 라벨 | `--font-serif-en` | `400` (Italic) | `12px` ~ `13px` | `1.35` | `0.3px` ~ `0.5px` |
| **UI. 제어 요소** | 버튼 글자, 메인 배지, 카드 링크 텍스트 | `--font-sans-en` | `600` | `14.5px` (배지 `11.5px`) | `1.0` ~ `normal`| `0.1px` ~ `0.3px` |

## 3. 타이포그래피 구현 원칙

1. **상속과 토큰화**:
   - 개별 엘리먼트 스타일에 `font-family: 'DM Sans'` 혹은 `font-family: 'Outfit'` 과 같은 폰트 이름을 하드코딩하지 마십시오.
   - 반드시 `:root`에 정의된 `--font-serif-ko`, `--font-serif-en`, `--font-sans-en`, `--font-body` 토큰을 통해 상속되도록 설정해야 브라우저의 폰트 교체 및 최적화가 유연해집니다.
2. **이질감 방지 (Font Matching)**:
   - 한국어와 영어가 함께 혼용될 때 자간과 높낮이가 불규칙해지는 현상을 막기 위해, 국문 텍스트에는 `--font-serif-ko` 혹은 `--font-body`를 우선 배치하고, 기하학적 수치나 영문 고유 지시어에는 `--font-sans-en`을 적극 활용합니다.
3. **가독성을 위한 비례 관계**:
   - 폰트의 크기가 커질수록 자간(Letter Spacing)은 더욱 조밀해져야 텍스트의 파편화가 일어나지 않습니다 (예: `60px` 크기에 `-2px` 자간 적용).
   - 본문 텍스트의 크기가 작아질수록 피로도를 낮추기 위해 행간(Line Height)을 점진적으로 넓혀줍니다 (예: `13px` 크기에 `1.7` 행간 적용).
